# Choose Your Tool

What do you want to do with BODS data?

| Goal | Tool |
|---|---|
| Validate a dataset against the standard | [bods-validator](https://github.com/StephenAbbott/bods-validator) |
| Query ownership graphs and control chains | [bods-neo4j](https://github.com/StephenAbbott/bods-neo4j) |
| Link with sanctions / PEP data | [bods-ftm](https://github.com/StephenAbbott/bods-ftm) |
| Run large-scale SQL analytics | [bods-gql](https://github.com/StephenAbbott/bods-gql) |
| Work in spreadsheets or Pandas | [bodsdata](https://github.com/openownership/bodsdata) |
| Build knowledge graphs, use SPARQL | [bodsld](https://github.com/openownership/bodsld) |
| Exchange with legacy XML systems | [bods-xml](https://github.com/StephenAbbott/bods-xml) |
| Collect data via web forms | [bods-forms](https://github.com/StephenAbbott/bods-forms) |

Still unsure? Start with validation — every other workflow assumes a clean dataset.
