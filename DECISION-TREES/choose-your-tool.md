# Choose Your Tool

What do you want to do with BODS data?

| Goal | Tool |
|---|---|
| Validate a dataset against the standard | [bods-validator](https://github.com/StephenAbbott/bods-validator) |
| Test a BODS adapter against canonical fixtures | [bods-fixtures](https://github.com/StephenAbbott/bods-fixtures) / [pytest-bods-fixtures](https://github.com/StephenAbbott/pytest-bods-fixtures) |
| Collect data via web forms | [bods-forms](https://github.com/StephenAbbott/bods-forms) |
| Query ownership graphs and control chains | [bods-neo4j](https://github.com/StephenAbbott/bods-neo4j) |
| Link with sanctions / PEP data | [bods-ftm](https://github.com/StephenAbbott/bods-ftm) |
| Run large-scale SQL analytics on BigQuery | [bods-gql](https://github.com/StephenAbbott/bods-gql) |
| Use a columnar format for ML / vector workloads | [bods-lance](https://github.com/StephenAbbott/bods-lance) |
| Exchange with legacy XML systems | [bods-xml](https://github.com/StephenAbbott/bods-xml) |
| Feed BODS into Google AML AI | [bods-aml-ai](https://github.com/StephenAbbott/bods-aml-ai) |
| Work with ICIJ Offshore Leaks data | [bods-icij-offshoreleaks](https://github.com/StephenAbbott/bods-icij-offshoreleaks) |
| Import OpenCorporates relationship data | [bods-opencorporates](https://github.com/StephenAbbott/bods-opencorporates) |
| Import BrightQuery ownership data | [bods-brightquery](https://github.com/StephenAbbott/bods-brightquery) |
| Import Kyckr relationship data | [bods-kyckr](https://github.com/StephenAbbott/bods-kyckr) |
| Work in spreadsheets, CSV, SQLite, Postgres or Parquet | [bodsdata](https://github.com/openownership/bodsdata) |
| Analyse published BODS data in notebooks | [bodsanalysis](https://github.com/openownership/bodsanalysis) |
| Build knowledge graphs, use SPARQL | [bodsld](https://github.com/openownership/bodsld) |
| Explore RDF-based risk detection patterns | [bodsriskdetection](https://github.com/openownership/bodsriskdetection) |

Still unsure? Start with validation — every other workflow assumes a clean dataset.
