# Tools

Mappings, converters, and utilities for working with BODS data.

This folder will host:

- Schema mappings (BODS → Neo4j, BODS → FTM, BODS → RDF, etc.)
- Conversion scripts and examples
- Utility snippets for common operations

Contributions welcome. See the top-level [README](../README.md) for the current tool inventory and links to standalone repositories.
