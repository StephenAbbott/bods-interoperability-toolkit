- [FollowTheMoney](https://followthemoney.tech/) — entity-centric data model used by OCCRP / OpenSanctions
- [OpenSanctions](https://www.opensanctions.org/) — consolidated sanctions and PEP data
- [Neo4j](https://neo4j.com/) — property graph database
- [RDF / SPARQL](https://www.w3.org/RDF/) — W3C linked data standards
