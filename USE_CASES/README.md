# Use Cases

Real-world scenarios where BODS interoperability matters.

Planned content:

- **Procurement screening** — checking suppliers against beneficial ownership data
- **AML / KYB pipelines** — consuming BODS in compliance workflows
- **Investigative journalism** — tracing cross-border ownership
- **Registry publication** — producing BODS output from national registries
- **Sanctions and PEP linking** — joining BODS with OpenSanctions / FTM data

Each use case will cover the problem, the BODS data required, the conversion path, and an end-to-end worked example.
