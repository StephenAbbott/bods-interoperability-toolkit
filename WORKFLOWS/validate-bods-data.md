# Validate BODS Data

## When to use this
- Before publishing data  
- Before converting formats  
- When debugging data issues  

## Steps
1. Run a BODS validator  
2. Identify schema errors  
3. Fix structure and revalidate  

## Common issues
- Missing statements  
- Invalid ownership links  
- Schema mismatch  

## Output
A clean, valid BODS dataset ready for use
